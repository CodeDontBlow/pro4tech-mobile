# Orbita - Mobile

Esse repositório tem como objetivo  abrigar o código da interface Mobile do projeto Orbita, com visualização para clientes e atendentes.

## Estrutura do Projeto

``` markdown
orbita-mobile/
├── app/                   # Telas 
│  └──  (tabs)/            # Telas acessíveis por tabs
├── assets/                # Arquivos estáticos
├── components/            # Componentes criados
├── components/            # Componentes criados
├── hooks/                 # Hooks criados para o projeto
├── package-lock.json      # Dependências do projeto
├── package.json           # Dependências do projeto
└──  README.md             # Este arquivo
```

## Rodando o Projeto

``` markdown
   # Em dispositivos Android
   npm run android      

   # Em dispositivos IOS
   npm run ios

   # No seu navegador
   npm run web
```
