# Orbita - Mobile

Esse repositório tem como objetivo abrigar o código da interface Mobile do projeto Orbita, um aplicativo de chamados de suporte com visualização para clientes e atendentes.


## Estrutura do Projeto

``` markdown
orbita-mobile/
├── app/                   # Telas e navegação
│  └──  (tabs)/            # Telas acessíveis por tabs
├── assets/                # Arquivos estáticos
├── components/            # Componentes criados
├── constants/             # Valores globais (cores, etc.)
├── hooks/                 # Hooks criados para o projeto
├── package.json           # Dependências do projeto
└──  README.md             # Este arquivo
```

## Instalação

``` markdown
   # Clone o repositório
   git clone https://github.com/seu-usuario/pro4tech-mobile.git
   
   # Instale as dependências
   npm install
```

## Rodando o Projeto

``` markdown
   # Em dispositivos/emulador Android
   npm run android      

   # Em dispositivos/emulador iOS
   npm run ios

   # No seu navegador
   npm run web
```
